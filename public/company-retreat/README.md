## 手順

1. 上記のファイル名で画像を用意する
2. このフォルダ（`public/company-retreat/`）に配置する
3. ビルドして反映される

## 注意事項

- **ファイル名は完全一致**させてください（大文字・小文字も区別されます）
- 画像が見つからない場合は、自動的にサンプル画像が表示されます
- JPG / PNG / WebP 形式に対応しています（拡張子を変更する場合は `src/pages/CompanyRetreat.tsx` の `PHOTOS` 定数も変更してください）
